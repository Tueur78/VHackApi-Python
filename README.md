External dependencies you'll need include:
- requests
- pytesseract
- Pillow

 
 
To install these on Linux:
```sh
$ pip install requests  
$ pip install pytesseract 
$ pip install futures  
$ pip install Pillow
or  
$ pip install tesseract
``` 

For launching on Linux distros:  
```sh
$ rm -rf console.py
$ mv console_linux.py console.py 
$ python main.py 
or 
$ ./main.py
``` 
For Windows:
